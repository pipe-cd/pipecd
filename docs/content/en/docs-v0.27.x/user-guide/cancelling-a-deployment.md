  This page describes how to cancel a running deployment.
---

A running deployment can be cancelled from web UI at the deployment details page.

If the application rollback is enabled in the application configuration, the rollback process will be executed after the cancelling. You can also explicitly specify to rollback after the cancelling or not from the web UI by clicking on `â–¼` mark on the right side of the `CANCEL` button to select your option.
